# The Tie That Binds - Archive

A digital archive of "The Tie That Binds" newsletters and related historical documents from the Titus family. This application provides a user-friendly interface to browse and view PDF copies of newsletters ranging from 1975 to 2010.

## Features

- **Chronological Archive**: Browse newsletters and documents sorted by date.
- **PDF Viewer**: View documents directly within the application without downloading.
- **Responsive Design**: Optimized for both desktop and mobile viewing.
- **Support**: Integrated donation option to help maintain the archive.

## Tech Stack

- **Frontend Framework**: React
- **Build Tool**: Vite
- **UI Library**: Chakra UI
- **Language**: TypeScript

## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/Jchronister2/The-Tie-That-Binds.git
   ```

2. Navigate to the project directory:

   ```bash
   cd The-Tie-That-Binds
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

### Development

To start the development server:

```bash
npm run dev
```

### Build

To build the application for production:

```bash
npm run build
```

To preview the production build locally:

```bash
npm run preview
```

## Deployment

This project is configured to deploy automatically to GitHub Pages using GitHub Actions when changes are pushed to the `master` branch.
